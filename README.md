# scene-pan
Smooth scene panning on mouse/touch drag

example
===
https://codepen.io/jackismissing/pen/vJmpWO

usage
====

Import
---
```
import ScenePan from 'scene-pan';
```

Instantiation
---
```
const scenePan = new ScenePan({
    el: document.querySelector('.pan-wrapper'),
    ease: .08
});
```

Options
---
ease: default to 0.8;
